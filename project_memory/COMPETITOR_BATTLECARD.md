## 差別化の核心（3つの独自性）

```

```

```
PPALコース受講 → LINE自動フォロー → 購買タイミング最適化
他のフォークにない「教育コース × CRM」の統合
```

### 3. AI診断シナリオ（みやびAI）
```
7問診断 → プロダクトマッチング → 7日ステップ配信
Claude APIで高品質な自動応答 + パーソナライズ配信
```
